Welcome to the Network

AnyfestoCHIP is CHIP based local networking content server . It is in the pocket of someone somwhere close to you.

AnyfestoCHIP offers ebooks, files and many hours of audio as shared files and as streaming audio.
To see what audio is being shared please head on over to the Files section.
If you would like to listen to the audio being streamed live please have your audio streaming app of choice point to 10.11.99.1 port 8085. A popular app that runs on Linux, Windows, Mac, Android and iOS is VLC.
This is an off-line network, there is no connection to the greater Internet. No activity is being loged

About AnyfestoCHIP

AnyfestoCHIP is a handy CHIP based server that fits in a pocket. It is battery powered and uses wifi to let others connect to it. AnyfestoCHIP uses Anyfesto, a variant of Library/PirateBox, to serve up ebooks and files via a web interface plus audio streaming via VLC. Inspired by pirate radio and the free culture movement, it uses Free, Libre and Open Source software (FLOSS) to create a mobile wireless file sharing network where users can anonymously share images, video, audio, documents, and other digital content.

Anyfesto is designed to be safe and secure. No logins are required and no user data is logged. The system is purposely not connected to the Internet in order to prevent tracking and preserve user privacy. AnyfestoCHIP is built using LibraryBox, PirateBox, VLC, Lighttp, and runs on Debian Linux

If you would like to build your own AnyfestoCHIP go to this page on GitHub to learn how.
